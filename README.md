# 🎓 GPA Calculator & To-Do List App

A powerful Flutter application that helps students track their academic performance and manage daily tasks efficiently. Built with Flutter and Dart, this app provides an intuitive interface for GPA calculation and task management.

## ✨ Features

### 📊 GPA Calculator
- **Semester Management**: Enter academic year and select semester (1-8)
- **Course Management**: Add courses with:
  - Course name
  - Credit hours (1 or 2 only)
  - Letter grades (A+ to D/ESA)
- **Accurate GPA Calculation**: Uses standard formula
- **Calculation Breakdown**: Shows step-by-step calculation
- **Real-time Updates**: GPA updates as you add/remove courses

### ✅ To-Do List
- **Task Management**: Add, complete, and delete tasks
- **Status Tracking**: Mark tasks as complete/incomplete
- **Timestamp**: Shows when each task was created
- **Visual Feedback**: Completed tasks show strikethrough and green background
- **Clean Interface**: Easy-to-use task management
